# Chandraniv Chakraborty — Portfolio

## 📁 FOLDER STRUCTURE

```
portfolio/
│
├── index.html          ← Main page (DO NOT rename)
├── style.css           ← All styling
├── script.js           ← All animations & JS
│
└── assets/
    ├── images/
    │   └── profile.jpg         ← YOUR PHOTO (square crop, any size)
    │
    ├── resume/
    │   └── resume.pdf          ← YOUR RESUME PDF
    │
    ├── certificates/
    │   ├── certificate-1.pdf   ← Your cert (update href in HTML too)
    │   └── certificate-2.pdf
    │
    └── marksheets/
        ├── marksheet-sem1.pdf  ← Semester 1 grade card
        ├── marksheet-sem2.pdf  ← Semester 2 grade card
        └── marksheet-sem3.pdf  ← Semester 3 grade card
```

## ✅ WHAT TO UPDATE IN index.html

1. **Your email** — Search `your@email.com` → replace with real email
2. **GitHub link** — Search `https://github.com/` → your profile URL
3. **LinkedIn link** — Search `https://linkedin.com/` → your profile URL
4. **LeetCode link** — Search `https://leetcode.com/` → your profile URL
5. **Certificate titles & issuers** — Edit the cert-card sections
6. **Project GitHub/demo links** — Replace the `href="#"` in project cards

## 🚀 HOW TO RUN LOCALLY

Just open `index.html` with **Live Server** in VS Code.
- Right-click `index.html` → "Open with Live Server"

## 🌐 GITHUB PAGES DEPLOY

1. Push entire folder to GitHub repo
2. Settings → Pages → Source: `main` branch, `/ (root)`
3. Live at: `https://yourusername.github.io/portfolio`
